HTML Tags used in the Assignment

  • HTML: this tag is the container for all other HTML elements
  • Head: this element is a container for metadata (data about data) and is placed between the html tag and the body tag.
  • Title: this tag defines the title of the document. It is very important for SEO.
  • Body: this tag defines the document's body.
  • div: this tag defines a division or a section in an HTML document.
  • headers: these tags are used to define HTML headings. They range from 1 to 6.
  • a: this tag defines a hyperlink, which is used to link from one page to another.
  • button: this tag defines a clickable button.
